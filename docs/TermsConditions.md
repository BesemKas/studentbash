### 5.1 QR Code Entry

- Each ticket includes a unique QR code
- QR codes are scanned at the event gate for validation
- Tickets are valid for **one-time use only**
- Once scanned and validated, the ticket is marked as used

### 5.2 Entry Requirements

To gain entry to an event, you must:
- Present a valid, verified ticket with QR code
- Present valid government-issued photo ID matching the ticket holder name
- Comply with age verification requirements
- Comply with all Event Organizer entry policies

### 5.3 Gate Validation Process

The gate validation system checks:
- Payment verification status
- Ticket validity and date (for day pass tickets)
- Age status (Adult/Minor)
- Whether the ticket has already been used

**Denial of Entry**: Entry will be denied if:
- Payment is not verified
- Ticket is not valid for the current event date (day pass)
- Ticket has already been used
- ID does not match ticket holder information
- Age restrictions are violated
